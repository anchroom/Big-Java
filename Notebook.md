### 1.6 Errors

#### Compile-time errors = Syntax errors

Wenn der Compiler eine oder mehrerer Fehler findet, die kein Program in die Java Instruktion übersetzen kann, werde er einfach stoppen und aufhören.

#### Run-time errors = Logic errors

Das Program gibt nicht aus, was es ausführen soll. Es gibt kein error message aus. 

### 1.7 Wie kann man ein Algorithm durch Pseudocode beschreiben

1. Eingaben und Ausgaben bestimmen;

2. Trennen das Problem in den kleine Aufgabe;

3. Beschreiben jede Aufgabe mit Pseudocode;

4. Test Pseudocode
